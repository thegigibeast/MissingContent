Missing Content: Suggestions is an open-source mod that aims to implement content from various player suggestions. 

Here is a list of content that has been implemented for now (note that there might be some tweaks to those suggestions in order to balance the content). Content can include items, NPCs, mechanics and much more:
* https://forums.terraria.org/index.php?threads/the-heated-mirror.80463/
* https://forums.terraria.org/index.php?threads/allowing-music-boxes-to-play-in-vanity-slots.9402/
* https://forums.terraria.org/index.php?threads/pumpkin-tools.13470/
* https://forums.terraria.org/index.php?threads/cursed-bone-charm-and-snake-eyes.5347/
* https://forums.terraria.org/index.php?threads/late-game-enemies-and-their-lack-of-debuffs.1507/
* https://forums.terraria.org/index.php?threads/debuff-resistance-improvements-and-changes-to-immunity-items.1502/
